# 工程实践手记（个人）

本目录放置 **个人向** 内容，与 [`toolbox/engineering/`](../../../toolbox/engineering/README.md) 中的参考体系配合使用：

- **落地记录**：某一云账号/项目上的拓扑、参数选择、踩坑与复盘。
- **ADR（架构决策记录）**：重大选型背景、备选方案、取舍结论与日期。
- **演练笔记**：备份恢复、故障演练、发布变更记录摘要。

约定：`toolbox/engineering` 保持相对通用；此处可含敏感信息的 **脱敏版** 或不提交敏感段落。

## 子目录建议（按需创建）

| 目录 | 用途 |
|------|------|
| `adrs/` | 架构决策记录 |
| `runbooks/` | 个人值班与应急步骤补充 |
| `incidents/` | 事故复盘（脱敏） |
