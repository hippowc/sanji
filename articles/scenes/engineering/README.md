# 工程实践手记（个人）

本目录存放 **个人向、含上下文** 的内容，与 [`toolbox/engineering/`](../../../toolbox/engineering/README.md) 中的通用参考配合使用：

| 类型 | 放置内容 | 示例 |
|------|----------|------|
| **落地拓扑** | 具体云账号下的 VPC、命名规则、网段 | 「某公司 prod 网络一页纸」（脱敏） |
| **ADR** | 架构决策背景、备选方案、结论 | 「为何 staging 不上 Mesh」 |
| **Runbook 实例** | 与内部值班工具链接的步骤 | 「RDS 只读副本延迟预警处置」 |
| **事故复盘** | blameless postmortem（脱敏） | 时间线、根因、行动项 |

**约定：** `toolbox/engineering` 保持厂商中立与可复用；此处可写 **脱敏后的真实参数**。涉密内容勿提交公开仓库。

## 建议子目录（按需创建）

- `adrs/` — 架构决策记录  
- `runbooks/` — 个人或团队 Runbook 增补  
- `incidents/` — 事故复盘摘要
