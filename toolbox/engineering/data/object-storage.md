# 对象存储

## 1. 典型用途

| 用途 | 说明 |
|------|------|
| 用户上传 | 头像、附件；配合预签名 URL 直传减轻应用带宽 |
| 静态站点 | 前端构建产物 |
| 日志/备份归档 | 低成本存储类；取回延迟可较高 |

## 2. 权限模型

- **Bucket 默认私有**；公共读仅限确实公开的静态资源 bucket。  
- **列举（List）权限** 与 **读取（Get）** 分离管理。  
- **预签名 URL**：限时、限 HTTP 方法；服务端签发。

## 3. 生命周期与成本

- **Standard → IA / Glacier 类**：按访问频率降级。  
- **生命周期规则**：日志 30 天热 → 归档。  
- **清理未完成分片**：multipart 中断残留计费。

## 4. 一致性与 CDN

- **覆盖写**：注意 CDN 缓存失效策略（版本化文件名优先）。  
- **跨区域复制**：合规与延迟需求驱动。
