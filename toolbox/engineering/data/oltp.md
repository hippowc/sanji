# 关系型与 OLTP

> **状态：** 骨架提纲。

## 本文目标

**托管关系型数据库** 在最小生产中的选型维度、连接管理、备份与高可用基线。

## 建议撰写要点

- **选型维度**：PostgreSQL / MySQL 族、运维托管程度、扩展模式（读写分离延后）。
- **连接池**：应用侧池大小与实例规格关系；避免连接风暴。
- **迁移**：与 [迁移与发布](../cicd/db-migrations-and-release.md) 对齐。
- **备份**：自动备份、保留期、演练频率指针（链到运维卷）。
- **安全**：私网访问、最小权限账号、审计。
