# TLS、WAF 与限流

## 1. TLS

- **全程 HTTPS**：入口终结 TLS；内部东西向可按威胁模型决定是否 mTLS（网格场景）。  
- **证书**：托管 ACM/证书服务自动续期；禁止手工拷贝过期证书。  
- **协议与套件**：禁用老旧 TLS；HSTS（在确认无纯 HTTP 依赖后）。

## 2. WAF（Web 应用防火墙）

**价值：** OWASP Top 10 规则集、Geo 封锁、Bot 缓解（视厂商）。

**成本：** 误杀业务流量；需要 **日志采样调优** 与 bypass 流程。

**建议：** 对外 API 与站点在 **有一定攻击面或合规要求** 时启用托管 WAF；规则从监控模式起步。

## 3. 限流与防刷

| 层级 | 手段 |
|------|------|
| 边缘/API 网关 | 每 IP / 每 Key 配额 |
| 应用 | 业务级令牌桶（结合 Redis） |

与 **登录、验证码、短信** 接口重点防护。

## 4. DDoS

- **网络层**：托管边缘清洗（各云 Anti-DDoS）；隐藏源站 IP（仅暴露 LB）。  
- **应用层**：WAF + 限流 + 队列削峰。
