# 备份与恢复

## 1. 范围

| 资产 | 策略 |
|------|------|
| OLTP | 托管自动备份 + PITR（若可用） |
| 对象存储 | 版本控制防误删；跨区域复制（按需） |
| 配置 | IaC 与 Git；密钥单独备份策略 |
| 镜像 | 镜像仓库复制 |

## 2. RPO / RTO

在 [最小生产](../overview/minimal-production.md) 中与业务共同文档化：

- **RPO**：能接受丢失多久事务（决定备份频率与 binlog 保留）。  
- **RTO**：可停机多久（决定演练目标时间）。

## 3. 演练

- **频率**：至少季度一次 **恢复到隔离 VPC/实例**。  
- **记录**：实际耗时、权限卡点、脚本缺口。  
- **自动化**：一键恢复脚本在安全环境验证后再用于应急。

## 4. 与应用发布的关系

备份 **不能替代** 错误的迁移策略；迁移失败优先评估 **向前修复**。
