# 容量与成本

## 1. 容量信号

| 信号 | 可能动作 |
|------|----------|
| CPU 持续 >70% | 增加副本或规格 |
| 内存逼近限制 | 调整 limits 或排查泄漏 |
| P99 延迟上升 | 数据库慢查询 / 下游超时 |
| 队列积压 | 扩容消费者或优化处理逻辑 |

## 2. 自动伸缩

- **无状态服务**：HPA on CPU/RPS/custom metrics。  
- **计划伸缩**：已知促销活动窗口预先扩容。  
- **集群 autoscaler**：节点池跟随 Pod pending 扩容（注意冷启动延迟）。

## 3. 成本治理（入门）

- **标签（Tags）**：`env`、`service`、`owner`；便于账单分摊。  
- **闲置回收**：非 prod 环境定时关停；Review 超大实例。  
- **存储分层**：日志与归档生命周期。  
- **告警**：月度账单异常阈值。

## 4. FinOps 进阶（超出最小生产默认）

- 部门预算、单元经济（unit economics）、承诺用量折扣等——按需专题扩展。
