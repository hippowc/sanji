## 本文目标

## 建议收录条目（示例）

| 术语 | 说明 |
|------|------|
| SLI / SLO / SLA | 服务测量指标、目标、对外承诺的区别 |
| RPO / RTO | 恢复点目标、恢复时间目标 |
| OIDC / Workload Identity | CI/CD 与云平台对接的无长期密钥模式 |
| BFF | Backend for Frontend，面向移动端/Web 的聚合层 |
| 不可变基础设施 | 镜像/制品版本不可原地篡改的部署思想 |
| 灰度 / 金丝雀 / 蓝绿 | 发布策略概念对照 |
