# 卓越架构自检表（精简）

本表抽象自公有云常见的 **Well-Architected** 类支柱（不同厂商命名略有差异），用于 **上线前或季度复盘** 快速扫描；细则见各卷正文。

## 可靠性

- [ ] 多实例或自动恢复已启用；健康检查区分存活与就绪  
- [ ] 备份与恢复演练已记录 RPO/RTO  
- [ ] 依赖故障有降级或超时熔断策略  
- [ ] 变更可回滚；数据库变更有 expand-contract 思路  

## 安全

- [ ] 数据面默认私网；管理访问可审计  
- [ ] TLS 全覆盖用户入口；证书自动续期  
- [ ] IAM 最小权限；无共享长期云密钥  
- [ ] 密钥与配置不进镜像与 Git；泄漏响应流程已知  

## 性能效率

- [ ] 连接池与线程池尺寸与规格匹配  
- [ ] 静态资源走 CDN；缓存策略清晰  
- [ ] 数据库慢查询有监控；热点有缓存或读写分离评估  

## 成本优化

- [ ] 资源标签完备；非生产有定时关停或缩容  
- [ ] 日志与追踪保留策略控制成本  
- [ ] 闲置实例与磁盘定期复盘  

## 运维卓越

- [ ] CI/CD 门禁齐全；制品不可变  
- [ ] 日志/指标/告警可达；Runbook 覆盖高频故障  
- [ ] 值班与升级路径明确
