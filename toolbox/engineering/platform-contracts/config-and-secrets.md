# 配置、密钥与十二因子实践

> **状态：** 骨架提纲。

## 本文目标

说明公有云上 **配置与密钥分离**、不进库、不进镜像层的常规做法，以及与十二因子应用的对应关系。

## 建议撰写要点

- **配置分级**：本地默认值、共享测试配置、生产仅存在于密钥管理与运行时注入。
- **密钥**：开发机 `.env` 不入库；CI 用 OIDC；运行时用托管密钥服务挂载或 API 拉取。
- **十二因子**：无状态进程、端口绑定、管理进程等与本仓库运行时章节互链。
- **反模式**：把生产数据库口令编译进前端包；在日志中打印密钥。
