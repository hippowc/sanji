# 可观测性（开源）

指标、日志、链路追踪与采集器相关开源索引（**非全自动来自 Star**，可按需同步到你的 Star 列表）。

| 仓库 | 说明 |
|------|------|
| [prometheus/prometheus](https://github.com/prometheus/prometheus) | 指标与告警规则引擎 |
| [grafana/grafana](https://github.com/grafana/grafana) | 可视化与告警面板 |
| [VictoriaMetrics/VictoriaMetrics](https://github.com/VictoriaMetrics/VictoriaMetrics) | Prometheus 兼容的长期存储与查询引擎 |
| [grafana/loki](https://github.com/grafana/loki) | 日志聚合（label 索引模型） |
| [grafana/tempo](https://github.com/grafana/tempo) | 链路追踪后端 |
| [jaegertracing/jaeger](https://github.com/jaegertracing/jaeger) | 分布式追踪系统 |
| [open-telemetry/opentelemetry-collector](https://github.com/open-telemetry/opentelemetry-collector) | 厂商中立的采集与导出 |
| [vectordotdev/vector](https://github.com/vectordotdev/vector) | 高性能日志/指标管线 |
